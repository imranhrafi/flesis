# Flesis

Flesis is a file manager application built with Next.js and Tailwind CSS using the DaisyUI plugin. It provides a clean and modern user interface for efficiently managing files.

## Getting Started

This is a [Next.js](https://nextjs.org/) project bootstrapped with [`create-next-app`](https://github.com/vercel/next.js/tree/canary/packages/create-next-app).

### Prerequisites

To run the project locally, you need to have the following software installed on your machine:

- Node.js (v14 or higher)
- npm (v7 or higher) or Yarn

### Installation

1. Clone the repository:

   ```bash
   git clone https://github.com/imranhrafi/flesis.git
````

2. Navigate to the project directory:

   ```bash
   cd flesis
   ```

3. Install the dependencies:

   ```bash
   npm install
   ```

4. Start the development server:

   ```bash
   npm run dev
   ```

5. Open your browser and visit [http://localhost:3000](http://localhost:3000) to see the application running.

### Usage

The application provides a user-friendly interface for managing files efficiently. You can perform various file operations, such as uploading, downloading, renaming, and deleting files.
